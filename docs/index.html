Edukatus Analytics — Webbtillagg for Progress

Sammanfattning: Edukatus Analytics skickar aldrig data till nagra externa servrar. All data stannar i din webblasare, krypterad med AES-256, och ar bara tillganglig for dig.

1. Vilka data samlar tillbagget in?

Edukatus Analytics hamtar data fran Progress (progress.edukatus.se) med din befintliga inloggning. Tillbagget samlar in:

2. Var lagras data?

All data lagras lokalt i din webblasare med foljande sakerhetsatgarder:

  • AES-256-GCM-kryptering — all data i IndexedDB ar krypterad med en nyckel unik for din tillbaggsinstallation
  • Krypteringsnyckeln harleds fran ett slumpmassigt salt + tillbaggets ID via PBKDF2 (100 000 iterationer)
  • Ingen annan webbplats, tillagg eller app kan lasa den krypterade datan
  • Ingen data skickas nagonstans — inte till Edukatus, inte till nagon server, inte till nagon tredje part
Personnummer lagras aldrig. Tillbagget laser personnummer enbart for att berakna om eleven ar over 18 ar. Det haledda vardet (alder, myndig ja/nej) sparas — aldrig personnumret sjalvt.
Vardnadshavardata lagras aldrig pa disk. Kontaktuppgifter halls enbart i webblasarens arbetsminne under den aktiva sessionen och forsvinner nar webblasaren stangs.

4. Datadelning med tredje part

Ingen. Edukatus Analytics delar aldrig data med nagon tredje part. Tillbagget innehaller:

  • Ingen analyticskod (ingen Google Analytics, ingen telemetri)
  • Inga externa API-anrop (forutom till progress.edukatus.se)
  • Ingen reklam
  • Ingen sparing

5. Hur raderar jag data?

  • I tillbagget: Klicka "Rensa all data" i sidomenyn
  • Avinstallation: Nar du tar bort tillbagget raderas all lagrad data automatiskt
  • Vardnadshavardata: Raderas automatiskt nar du stanger webblasaren

6. GDPR och skolans ansvar

Edukatus Analytics behandlar samma data som du redan har tillgang till i Progress i din roll som larare/administratior. Tillbagget utvidgar inte din dataagang — det visualiserar och analyserar befintlig information.

Vi rekommenderar att:

  • Skolans dataskyddsombud (DPO) informeras om anvandningen
  • Tillbagget anvands pa skolans enheter med diskkryptering aktiverad
  • Data rensas regelbundet (t.ex. vid terminsslut)
